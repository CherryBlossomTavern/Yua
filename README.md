## Getting Started

#### `Installing Dependencies`
```npm
npm install
```
#### `Run in Dev Mode`
```npm
npm run dev
```
#### `Building`
```npm
npm run build
```
#### `Run In Production`
```npm
npm start
```
#### `Manual Lint`
```npm
npm run lint
```
#### `Manual Lint, Auto Fix Issues`
```npm
npm run lint:fix
```
